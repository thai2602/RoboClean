# PROJECT ROADMAP
# RoboClean Landing Page

## Overview

Dự án được chia thành 7 giai đoạn nhằm đảm bảo việc phát triển theo hướng tuần tự, dễ kiểm soát và có thể kiểm thử sau mỗi mốc hoàn thành.

Mỗi giai đoạn đều có:

- Objective
- Tasks
- Deliverables
- Definition of Done (DoD)

---

# Phase 1 — Project Foundation

## Objective

Thiết lập nền tảng kỹ thuật cho toàn bộ dự án.

## Tasks

### Frontend

- Initialize Next.js App Router
- Configure TypeScript
- Configure Tailwind CSS
- Configure ESLint + Prettier
- Setup Folder Structure

### Backend

- Initialize Spring Boot Project
- Configure Maven
- Configure Spring Data JPA
- Configure Validation
- Configure H2 Database

### Project

- Initialize Git Repository
- Setup Monorepo Structure
- Create Environment Variables

## Deliverables

- Frontend project
- Backend project
- Git repository
- Base folder structure

## Definition of Done

- FE & BE chạy thành công.
- Không còn lỗi build.
- Commit đầu tiên hoàn tất.
